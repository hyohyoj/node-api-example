## Deploy
[cloudtype](https://cloudtype.io/)을 이용하여 API 서버 배포

버전 : 16


### Install command
```Bash
npm ci --production
```
모듈에 대한 자세한 설명이 적혀있는 **package-lock.json**를 기반으로 설치한다는 명령어

### Start command
```Bash
node index.js
```
